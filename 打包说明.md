# 打包 Windows EXE 文件指南

## 快速打包（Windows 环境）

### 方法一：双击运行批处理脚本
1. 在 Windows 电脑上打开项目文件夹
2. 双击运行 `build_windows.bat`
3. 等待打包完成（约 2-5 分钟）
4. 生成的 EXE 文件在 `dist\加密Excel合并工具.exe`

### 方法二：命令行打包
```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 安装项目依赖
pip install -r requirements.txt

# 3. 执行打包
pyinstaller --name "加密Excel合并工具" ^
    --windowed ^
    --onefile ^
    --hidden-import openpyxl ^
    --hidden-import olefile ^
    --hidden-import cryptography ^
    --hidden-import msoffcrypto ^
    --hidden-import PyQt6 ^
    --hidden-import PyQt6.sip ^
    --exclude-module tkinter ^
    --exclude-module unittest ^
    --noconfirm ^
    main.py
```

## 参数说明

| 参数 | 说明 |
|------|------|
| `--windowed` | 隐藏命令行窗口，只显示 GUI |
| `--onefile` | 打包为单个 exe 文件 |
| `--hidden-import` | 显式包含依赖库 |
| `--exclude-module` | 排除不需要的模块，减小体积 |
| `--noconfirm` | 覆盖已有输出，无需确认 |

## 常见问题

### Q1: 打包后 exe 文件很大（100MB+）？
**A:** 这是正常现象。PyInstaller 会将 Python 运行时和所有依赖库打包进去。
- 正常大小：约 80-150 MB
- 如需减小体积，可尝试使用 UPX 压缩（需安装 upx.exe）

### Q2: 运行时提示缺少模块？
**A:** 在打包命令中添加 `--hidden-import` 参数：
```bash
--hidden-import 缺少的模块名
```

### Q3: 杀毒软件误报？
**A:** PyInstaller 打包的 exe 可能被某些杀毒软件误报。
- 解决方案 1：将 exe 添加到杀毒软件白名单
- 解决方案 2：使用代码签名证书签名（企业环境）
- 解决方案 3：改用 `--onedir` 模式（生成文件夹而非单个文件）

### Q4: 在 macOS/Linux 上能打包 Windows exe 吗？
**A:** 不能直接打包。需要：
- 方案 1：在 Windows 电脑上打包
- 方案 2：使用虚拟机或 Docker
- 方案 3：使用交叉编译工具（不推荐，复杂且不稳定）

### Q5: 如何添加应用图标？
**A:** 准备一个 .ico 文件，然后添加 `--icon` 参数：
```bash
pyinstaller ... --icon=app.ico main.py
```

## 分发说明

### 单个 exe 文件（--onefile）
- 优点：只有一个文件，方便分发
- 缺点：启动较慢（需解压到临时目录）
- 适用：小型工具、临时使用

### 文件夹模式（--onedir）
```bash
pyinstaller --name "加密Excel合并工具" --windowed --onedir main.py
```
- 优点：启动快，占用内存小
- 缺点：需分发整个文件夹
- 适用：频繁使用、性能要求高

## 更新版本

修改代码后重新打包：
```bash
# 删除旧的打包文件
rmdir /s /q dist build
del *.spec

# 重新打包
build_windows.bat
```

## 高级选项

### 使用 PyInstaller spec 文件
```bash
pyinstaller build.spec
```

### 包含额外文件
```bash
--add-data "配置文件.json;."
--add-data "模板文件夹;templates"
```

### 优化启动速度
```bash
# 使用 --onedir 模式
pyinstaller --onedir ...

# 或使用 PyInstaller 的压缩优化
pyinstaller --onefile --compress=2 ...
```
